# Python Mini Projects Pack

A collection of five beginner friendly Python projects built to practice
Python fundamentals, user input handling, APIs, file I/O, and basic game logic.

These projects were completed as hands on learning exercises to gain practical
experience reading, debugging, and extending Python code.

## Projects Included
- Weather App (API based, Celsius/Fahrenheit support)
- Stock Tracker (financial data via yfinance)
- Calculator (basic and scientific operations)
- File Parser (text analysis and word frequency)
- Tic Tac Toe (Human vs Human and Human vs Computer)

## Skills Demonstrated
- Python fundamentals
- Functions and control flow
- API consumption
- Error handling
- User input validation
- Basic data processing
- Simple game logic

## How to Run
Each project can be run individually from the repository root:

```bash
python <project_folder>/<script_name>.py

## Example:
python weather/weather.py
