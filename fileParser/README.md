A simple file parsing script that reads a text file, cleans the contents, and
reports basic statistics.

## Features
- Reads text from a file
- Removes punctuation and normalizes text
- Counts total number of words
- Identifies the most frequent word

## Files
- `parser.py`: main script
- `sample.txt`: example input file

## Skills Used
- File I/O
- String manipulation
- Dictionaries
- Error handling

## How to Run
```bash
python file_parser.py
