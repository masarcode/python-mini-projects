A command-line stock tracker that fetches and displays real-time financial
data using the yfinance API.

## Features
- Current stock price
- Daily high and low
- Market capitalization (formatted in M/B/T)
- Forward P/E ratio
- Dividend yield (normalized and formatted)

## Skills Used
- API consumption
- Data validation and formatting
- Financial data interpretation
- Defensive programming

## Dependencies
- yfinance

## How to run
python stock_tracker.py

Install with:
```bash
pip install yfinance
