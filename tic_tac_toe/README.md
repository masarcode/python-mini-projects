A command line Tic Tac Toe game supporting both human vs human and human vs
computer gameplay.

## Features
- Human vs Human mode
- Human vs Computer mode (random AI)
- Numbered board for easy input
- Win and tie detection
- Input validation

## Skills Used
- Game state management
- Control flow
- Randomized decision making
- User experience design

## How to Run
```bash
python tic_tac_toe.py
