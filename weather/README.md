A command-line weather application that retrieves current weather data from
the wttr.in API.

## Features
- City-based weather lookup
- Celsius or Fahrenheit temperature selection
- Current condition description
- Humidity display

## Skills Used
- HTTP requests
- API parsing
- User input validation
- Conditional logic

## Dependencies
- requests

## How to run
python weather_api.py

Install with:
```bash
pip install requests
